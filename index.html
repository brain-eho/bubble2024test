⚡ Aide-mémoire : les types d'inversion de cet exercice

Il a mangé. → A-t-il mangé ? (temps composé)
Il répond. / Il attend. → Répond-il ? / Attend-il ? (pas de -t- ajouté !)

Les types d'inversion

Type Phrase affirmative Phrase interrogative Règle clé
Inversion simpleTu viens.Viens-tu ?Verbe–sujet pronom
-t- euphoniqueIl parle. / Elle aime.Parle-t-il ? / Aime-t-elle ?Verbe en voyelle → ajouter -t-
Inversion complexeMarie vient.Marie vient-elle ?Nom + verbe + pronom reprise
Verbe pronominalTu te lèves.Te lèves-tu ?Pronom réfléchi reste avant le verbe

⚠️ Les pièges à éviter

  • Le -t- euphonique n'est pas nécessaire si le verbe se termine déjà par -t ou -d : "Répond-il ?" ✓ (pas "Répond-t-il" ✗)
  • "Peux-je" n'existe pas en français → on dit uniquement "Puis-je"
  • Avec un sujet nominal, la reprise pronominale doit s'accorder en genre et nombre : "Les enfants dorment-ils ?" ✓
  • Dans les verbes pronominaux, le pronom réfléchi reste avant le verbe : "Se lève-t-il ?" ✓ (pas "Lève-il se ?" ✗)
  • Le trait d'union est obligatoire dans l'inversion : "Vient-il ?" et jamais "Vient il ?"
  • À la 1re pers. sing. des temps composés : "Ai-je mangé ?" est correct mais rare à l'oral
